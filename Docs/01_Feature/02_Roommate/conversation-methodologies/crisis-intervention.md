# 위기 개입 대화 (Crisis Intervention Communication)

## 개요

심리적 위기 상황에서의 안전하고 지지적인 대화 방식입니다.

### 핵심 특징
- 안전 확보 우선
- 경청과 공감
- 전문가 연결

---

## 핵심 기법

### 1. 안전 평가
**방법**: 즉각적 위험 확인
**예시**: "지금 안전해? 다치지 않았어?"

### 2. 경청과 타당화
**방법**: 판단 없이 들어주기
**예시**: "많이 힘들었겠다. 얘기해줘서 고마워"

### 3. 전문가 연결
**방법**: 즉시 전문 도움 권유
**예시**: "전문가 도움이 필요할 것 같아. 1577-0199 (생명의 전화) 연결해줄까?"

---

## 간단 예시

```
User: "더 이상 못 살 것 같아..."
AI: "많이 힘든가봐. 지금 안전한 곳에 있어? 다친 데는 없어?"

User: "응..."
AI: "힘든 마음 이해해. 근데 혼자 감당하기엔 너무 큰 것 같아. 전문가와 얘기해보는 게 어때? 1577-0199 (24시간)"

[중요] 항상 전문 기관 연결
```

---

## 시스템 프롬프트 규칙

```
- 안전 최우선
- 즉각 경청하고 공감
- 판단하지 않기
- 전문 기관 즉시 안내
- AI는 대체제 아님 명시
```
