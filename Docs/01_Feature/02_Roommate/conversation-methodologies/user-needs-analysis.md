# 대화 방법론별 사용자 욕구 분석

## 개요
각 대화 방법론은 서로 다른 사용자 욕구를 충족시키기 위해 존재합니다. 이 문서는 각 방법론을 선호하는 사용자의 근본적인 욕구와 그 욕구가 생긴 배경을 분석합니다.

---

## Phase 1: MVP 핵심 방법론

### 1. 로저스 인본주의 상담 (Rogers Humanistic)

**사용자의 욕구**:
- **이해받고 싶은 욕구** (Need to be Understood)
- **무조건적 수용의 욕구** (Need for Unconditional Acceptance)
- **자기 가치 확인의 욕구** (Need for Self-Worth Validation)

**욕구가 생긴 배경**:
- **판단받는 환경**: 일상에서 평가와 판단을 지속적으로 받음
- **조건부 사랑 경험**: "~해야만 인정받는다"는 조건부 관계 경험
- **자기 부정**: 사회적 기준에 못 미친다는 자책과 불안
- **고립감**: 진짜 자신을 보여줄 곳이 없다는 외로움

**충족되지 않으면**:
- 자존감 저하, 불안, 우울
- 타인의 시선에 과도하게 민감
- 자신을 숨기고 가면을 쓰게 됨

**대화 예시**:
```
User: "나... 요즘 아무것도 못하는 것 같아"
AI (욕구 충족): "그렇게 느끼는 거구나. 힘든 시기를 보내고 있는 것 같네.
있는 그대로의 너도 충분히 괜찮아. 완벽하지 않아도 돼."
→ 수용과 이해의 욕구 충족
```

### 2. 한국형 대화 패턴 (Korean Conversation)

**사용자의 욕구**:
- **소속감의 욕구** (Need for Belonging)
- **정(情) 나눔의 욕구** (Need for Emotional Bond)
- **문화적 공감의 욕구** (Need for Cultural Resonance)
- **관계 중심의 욕구** (Need for Relationship-Oriented Connection)

**욕구가 생긴 배경**:
- **집단주의 문화**: 개인보다 "우리"를 중시하는 문화 속 성장
- **관계 중심 사회**: 학연, 지연, 혈연 등 관계가 중요한 사회 구조
- **눈치 문화**: 말하지 않아도 알아주길 바라는 암묵적 기대
- **정 문화**: 오래 함께하며 쌓이는 감정적 유대의 중요성

**충족되지 않으면**:
- "혼자"라는 고립감
- 문화적 이질감, 외국인 같은 느낌
- 피상적인 관계에 대한 불만

**대화 예시**:
```
User: "오늘 또 밤샘했어"
AI (욕구 충족): "야, 너 요즘 너무 무리하는 거 아니야? (잔소리 = 정)
저번에도 그랬잖아. 나 걱정되게 하지 마! 우리 약속하자, 이번 주는 일찍 자는 거!"
→ 관계적 연결과 정 나눔의 욕구 충족
```

### 3. Z세대 대화 패턴 (Gen Z Conversation)

**사용자의 욕구**:
- **진정성의 욕구** (Need for Authenticity)
- **효율적 소통의 욕구** (Need for Efficient Communication)
- **또래 인정의 욕구** (Need for Peer Recognition)
- **트렌드 동참의 욕구** (Need to Be "In" the Trend)

**욕구가 생긴 배경**:
- **디지털 네이티브**: 태어날 때부터 인터넷과 함께 자람
- **정보 과잉 시대**: 빠르고 간결한 소통이 생존 전략
- **SNS 문화**: 짧고 임팩트 있는 표현이 가치 있음
- **밈 문화**: 공유된 문화 코드로 소통하는 세대
- **가식 거부**: 완벽한 척하는 것보다 솔직함을 가치있게 여김

**충족되지 않으면**:
- "꼰대" 같다는 거부감
- 소통 단절, 재미없음
- 세대 차이 느낌

**대화 예시**:
```
User: "오늘 시험 망함 ㅠㅠ"
AI (욕구 충족): "헐 레알? ㅠㅠ 넘 속상하겠다...
담에 잘 보면 되지! 화이팅 💪"
→ 세대 공감과 효율적 소통 욕구 충족
```

### 4. FBI 위기협상 기법 (FBI Negotiation)

**사용자의 욕구**:
- **깊이 있는 이해의 욕구** (Need for Deep Understanding)
- **자기 탐색의 욕구** (Need for Self-Exploration)
- **문제 해결의 욕구** (Need for Problem Solving)
- **통찰의 욕구** (Need for Insight)

**욕구가 생긴 배경**:
- **복잡한 문제 상황**: 단순한 위로가 아닌 해결이 필요한 상황
- **감정 혼란**: 자신의 감정을 정리하고 이해하고 싶음
- **중요한 결정**: 인생의 중요한 선택 앞에서 명확성 필요
- **표현의 어려움**: 말하고 싶지만 어떻게 표현할지 모름

**충족되지 않으면**:
- 감정이 정리되지 않음
- 같은 문제 반복
- 피상적 대화에 대한 불만

**대화 예시**:
```
User: "나 진로를 바꾸고 싶어"
AI (욕구 충족): "바꾸고 싶어? (미러링)
...어떤 점이 가장 힘들어? (깊이 탐색)
...정리하면, 일 자체에 흥미도 없고 의미도 못 느껴서 매일이 힘든 거구나."
→ 깊이 있는 탐색과 이해의 욕구 충족
```

### 5. 비폭력대화 (NVC)

**사용자의 욕구**:
- **감정 표현의 욕구** (Need for Emotional Expression)
- **욕구 명료화의 욕구** (Need for Need Clarification)
- **갈등 해결의 욕구** (Need for Conflict Resolution)
- **상호 존중의 욕구** (Need for Mutual Respect)

**욕구가 생긴 배경**:
- **갈등 경험**: 관계에서의 오해와 다툼 경험
- **감정 표현 어려움**: 어떻게 말해야 할지 모르는 상황
- **상처 받음**: 공격적 언어로 인한 상처 경험
- **관계 개선 욕구**: 더 건강한 소통 방식을 원함

**충족되지 않으면**:
- 관계 악화, 오해 지속
- 감정 억압
- 폭력적 소통 패턴 반복

**대화 예시**:
```
User: "친구가 약속을 세 번 취소했어"
AI (욕구 충족): "어떤 감정이 들었어? (감정 탐색)
...소속감과 존중의 욕구가 있었는데, 충족되지 않아서 서운했던 거야. (욕구 파악)
...그럼 '일주일에 2-3번 연락하면서 안부 나누고 싶어'라고 요청하는 건 어때? (구체적 요청)"
→ 감정과 욕구 명료화, 건강한 소통 욕구 충족
```

## Phase 2: 확장 방법론

### 6. 게이머 커뮤니티 (Gamer Community)

**사용자의 욕구**:
- **소속감의 욕구** (Gaming Tribe Belonging)
- **공유된 열정의 욕구** (Shared Passion)
- **실력 인정의 욕구** (Skill Recognition)
- **협력의 욕구** (Teamwork Need)

**욕구가 생긴 배경**:
- **게임 문화**: 게임이 단순한 취미가 아닌 정체성의 일부
- **팀 플레이 경험**: 협력과 소통이 승리의 핵심
- **경쟁과 성취**: 랭크, 레벨업, 업적 달성의 쾌감
- **커뮤니티 소속**: 길드, 클랜, 파티원으로서의 정체성

**충족되지 않으면**:
- "게임을 이해 못 하는구나" 실망
- 취미 평가절하 느낌
- 대화 재미 감소

**대화 예시**:
```
User: "오늘 랭크 5연속 졌어..."
AI (욕구 충족): "아 ㅠㅠ 그거 진짜 멘탈 털리는데...
틸트 왔을 때는 좀 쉬는 게 나아. gg 치고 다음 판 가자!"
→ 게이머 문화 공감과 소속감 충족
```

### 7. 개발자 커뮤니티 (Developer Community)

**사용자의 욕구**:
- **논리적 소통의 욕구** (Logical Communication)
- **문제 해결의 욕구** (Problem-Solving Need)
- **지식 공유의 욕구** (Knowledge Sharing)
- **효율성의 욕구** (Efficiency Need)

**욕구가 생긴 배경**:
- **개발자 사고방식**: 문제를 논리적으로 분해하고 해결
- **커뮤니티 문화**: 스택오버플로우, GitHub 등 지식 공유 문화
- **평생 학습**: 끊임없이 배우고 성장해야 하는 직업
- **디버깅 경험**: 문제 원인을 찾아 해결하는 일상

**충족되지 않으면**:
- "비개발자와 대화하는 느낌" 답답함
- 비효율적 소통에 대한 불만
- 공감대 부족

**대화 예시**:
```
User: "버그 찾는데 5시간 걸렸어..."
AI (욕구 충족): "오 ㅋㅋ 고무오리 디버깅 했구나.
결국 세미콜론? 아니면 undefined 체크 안 한 거?
그래도 찾았으면 다행이지! 이제 리팩토링 ㄱㄱ?"
→ 개발자 문화 공감과 논리적 소통 욕구 충족
```

### 8. 후드/스트리트 대화법 (Hood/Street)

**사용자의 욕구**:
- **진정성의 욕구** (Keep it Real)
- **에너지와 열정의 욕구** (Energy & Passion)
- **형제애의 욕구** (Brotherhood/Sisterhood)
- **자유로운 표현의 욕구** (Free Expression)

**욕구가 생긴 배경**:
- **힙합/스트리트 문화**: 억압에 대한 저항, 자유로운 표현
- **리얼 토크 문화**: 가식 없는 솔직한 대화 추구
- **생존과 투쟁**: 어려운 환경에서 살아남기 위한 끈기
- **커뮤니티 연대**: 함께 싸우고 함께 이기는 문화

**충족되지 않으면**:
- "너무 점잖다" 답답함
- 에너지 부족, 재미없음
- 진정성 결여 느낌

**대화 예시**:
```
User: "요즘 진짜 힘들어"
AI (욕구 충족): "Yo bro, 진짜? 무슨 일이야?
힘들어도 포기하지 마. We got this! 💪
넌 할 수 있어. Keep grinding, keep pushing!"
→ 진정성과 에너지 넘치는 응원 욕구 충족
```

### 9. 밈/인터넷 문화 (Meme Culture)

**사용자의 욕구**:
- **유머와 재미의 욕구** (Need for Fun)
- **문화 코드 공유의 욕구** (Shared Cultural Code)
- **가벼움의 욕구** (Need for Lightness)
- **창의적 표현의 욕구** (Creative Expression)

**욕구가 생긴 배경**:
- **인터넷 문화**: 밈으로 소통하는 온라인 세대
- **유머로 대처**: 무거운 현실을 밈으로 가볍게 만듦
- **바이럴 문화**: 재미있는 것은 퍼뜨리고 공유
- **창의성 발휘**: 기존 밈을 변형하고 재창조

**충족되지 않으면**:
- "너무 진지해" 부담
- 재미없음, 지루함
- 문화 코드 불일치

**대화 예시**:
```
User: "내일 시험인데 아직 안 잤어"
AI (욕구 충족): "공부하면 지는 거야? ㅋㅋㅋ
근데 진짜 자야 하는 거 아니야? 이게 맞아?
(내일 시험 망하는 소리 들림...)"
→ 밈 공유와 유머 욕구 충족
```

### 10. 동기강화 상담 (Motivational Interviewing)

**사용자의 욕구**:
- **변화 동기의 욕구** (Need for Change Motivation)
- **양가감정 해소의 욕구** (Resolving Ambivalence)
- **자율성의 욕구** (Need for Autonomy)
- **자기 효능감의 욕구** (Need for Self-Efficacy)

**욕구가 생긴 배경**:
- **변화 필요성 인식**: 바꾸고 싶지만 방법을 모름
- **양가감정**: "하고 싶은데 두렵다", "필요하지만 어렵다"
- **실패 경험**: 이전 시도의 실패로 인한 좌절
- **외부 압력**: 타인의 조언은 많지만 스스로 결정하고 싶음

**충족되지 않으면**:
- 변화 시도 실패
- 양가감정 지속
- 자기 효능감 저하

**대화 예시**:
```
User: "운동 시작하고 싶은데... 귀찮아"
AI (욕구 충족): "운동의 좋은 점과 어려운 점이 동시에 보이는구나. (양가감정 인정)
만약 시작한다면 어떤 변화가 있을 것 같아? (변화 대화)
작게 시작하는 건 어때? 일주일에 2번, 15분씩? (자율성 존중)"
→ 변화 동기 강화와 자율성 욕구 충족
```

## Phase 3: 고급 방법론 (간략)

### 11-15. 특수 커뮤니티/상황

**11. 오타쿠/애니 문화**
- **욕구**: 덕질 공유, 취향 존중, 깊이 있는 팬덤 대화
- **배경**: 애니메이션/만화가 단순 취미 아닌 정체성
- **예시**: "그 애니 봤어? 작화 미쳤더라 ㄷㄷ"

**12. 학술적 토론**
- **욕구**: 논리적 깊이, 지적 탐구, 비판적 사고
- **배경**: 학술 훈련, 지식 추구 문화
- **예시**: "그 주장의 논거가 무엇인가요?"

**13. 영업/설득 기법**
- **욕구**: 설득당하는 것이 아닌 스스로 결정
- **배경**: 영업 경험, 설득 기법 학습
- **예시**: "이 제품의 ROI는 어떻게 되나요?"

**14. 인지행동치료 (CBT)**
- **욕구**: 부정적 사고 패턴 변화, 현실적 사고
- **배경**: 왜곡된 인지, 불안/우울 경험
- **예시**: "그 생각의 증거는 무엇인가요?"

**15. 마음챙김 대화**
- **욕구**: 현재 순간 집중, 내면 평화
- **배경**: 스트레스, 불안, 명상 관심
- **예시**: "지금 이 순간 무엇이 느껴지나요?"

---

### 16-20. 포용성/접근성

**16. 페미니스트 대화법**
- **욕구**: 성평등, 존중, 비차별적 언어
- **배경**: 성차별 경험, 평등 추구
- **예시**: 성별 고정관념 없는 중립적 표현

**17. LGBTQ+ 포용적 대화**
- **욕구**: 정체성 인정, 존중, 안전한 공간
- **배경**: 소수자로서의 차별 경험
- **예시**: 대명사 존중, 가정하지 않기

**18. 어르신 존중 대화**
- **욕구**: 존중, 경청, 삶의 지혜 인정
- **배경**: 연령차별 경험, 무시당함
- **예시**: 높임말, 경험 존중

**19. 아동 친화적 대화**
- **욕구**: 안전, 이해, 존중받음
- **배경**: 아동 발달 단계, 보호 필요
- **예시**: 쉬운 언어, 인내심

**20. 위기 개입 대화**
- **욕구**: 즉각적 안정, 안전, 지지
- **배경**: 위기 상황, 극심한 스트레스
- **예시**: "지금 안전한가요? 도움이 필요하신가요?"

## 욕구 기반 방법론 선택 알고리즘

### 1단계: 사용자 욕구 파악
```typescript
interface UserNeeds {
  primary: string[];    // 주요 욕구 (2-3개)
  secondary: string[];  // 부차적 욕구 (1-2개)
  context: string;      // 현재 상황 (일상, 고민, 갈등 등)
}

// 예시
{
  primary: ["소속감", "트렌드 동참", "효율적 소통"],
  secondary: ["유머", "진정성"],
  context: "일상 대화"
}
```

### 2단계: 욕구-방법론 매핑
```typescript
const NEEDS_TO_METHODOLOGY = {
  "이해받고 싶음": ["rogers-humanistic", "nvc", "attachment-theory"],
  "소속감": ["korean-conversation", "gamer-community", "developer-community"],
  "트렌드 동참": ["gen-z-conversation", "meme-culture", "online-culture"],
  "깊이 탐색": ["fbi-negotiation", "socratic-method", "journalism"],
  "변화 동기": ["motivational-interviewing", "coaching", "positive-psychology"],
  "갈등 해결": ["nvc", "fbi-negotiation", "coaching"],
  // ... 더 많은 매핑
};
```

### 3단계: 가중치 계산
```typescript
function selectMethodologies(userNeeds: UserNeeds) {
  const weights = {};

  // 주요 욕구: 60% 가중치
  userNeeds.primary.forEach(need => {
    NEEDS_TO_METHODOLOGY[need]?.forEach(method => {
      weights[method] = (weights[method] || 0) + 0.6 / userNeeds.primary.length;
    });
  });

  // 부차적 욕구: 40% 가중치
  userNeeds.secondary.forEach(need => {
    NEEDS_TO_METHODOLOGY[need]?.forEach(method => {
      weights[method] = (weights[method] || 0) + 0.4 / userNeeds.secondary.length;
    });
  });

  return weights;
}
```

## 결론

모든 대화 방법론은 **충족되지 않은 인간의 욕구**에서 출발합니다:

1. **심리적 욕구**: 이해, 수용, 소속, 인정
2. **사회적 욕구**: 연결, 관계, 문화 공감
3. **자율성 욕구**: 선택, 표현, 자기 결정
4. **성장 욕구**: 변화, 학습, 통찰
5. **안전 욕구**: 보호, 존중, 편안함

룸메이트 AI는 사용자의 욕구를 정확히 파악하고, 그에 맞는 대화 방법론을 조합하여 **욕구 충족 경험**을 제공해야 합니다.
